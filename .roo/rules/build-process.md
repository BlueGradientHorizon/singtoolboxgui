# Build Process

When testing if the project is buildable or running build commands:

- On Windows: Use `.\build.bat`
- On Linux/macOS: Use `./build.sh`
- On Termux: Use `./build_termux.sh`

The appropriate script should be selected based on the operating system context.

For APK builds specifically:
- On Windows: Use `.\build_apk.bat`
- On Linux/macOS/Termux: Use `./build_apk.sh`
